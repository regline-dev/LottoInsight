# 🚀 LottoInsight: 누구나 만드는 로또 번호 시각화기

"복잡한 코딩은 잊으세요! 재미로 만드는 나만의 로또 번호 생성 서비스입니다."

## 1. 이 프로젝트는요?
로또 번호를 그냥 뽑는 건 너무 심심하죠? 
내 꿈 내용을 넣으면 번호가 짠! 하고 나오고, 내가 뽑은 번호들이 어떤 패턴을 그리는지 눈으로 확인하는 재미있는 웹 서비스를 만드는 과정입니다. 누구나 쉽게 따라 하면서 나만의 서비스를 배포해보는 게 목표예요.

## 2. 이렇게 만들어요 (재미 요소)
- **10가지 골라 쓰는 재미:** 통계, 운세, 꿈 해몽 등 입맛대로 골라 번호를 뽑아요.
- **로또 패턴 한눈에:** 뽑힌 번호들이 로또 용지 위에서 어떤 모양을 만드는지 선으로 연결해 보여줍니다.
- **AI 덕담:** 번호 뽑고 끝? 아니죠! 행운을 빌어주는 그럴싸한 한 줄 덕담도 함께 나옵니다.

## 3. 우리 같이 해요! (사용한 기술)
어려운 서버 없이 아주 간단한 도구들만 씁니다.
- **Next.js & TypeScript:** 웹사이트의 뼈대입니다.
- **Tailwind CSS:** 예쁜 디자인을 쉽고 빠르게 입혀요.
- **Vercel:** 만든 사이트를 인터넷에 공짜로 1분 만에 올립니다.
